# CSV-Combiner
A small tool to merge and combine several CSV files into one CSV file.

It keeps duplicate rows, but it does filter out duplicate column headers if they share the exact same header name.


--------------- How To Use The Tool: ---------------
----------------------------------------------------
1- Press the "1- Select Files" button to choose the CSV files you want to combine.
\
\
2- From the upper box in the tool, left click with your mouse on the blue underlined column headers you want to keep.
\
2.1- Left click again to remove!
\
2.2- The 2nd box in the tool contains column headers you picked so far.
\
\
3- Press the "3- Start Combining!" button to combine the selected CSV files (using the column headers you picked earlier) into one CSV file.
