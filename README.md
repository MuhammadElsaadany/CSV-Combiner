# CSV-Combiner
A small tool to merge and combine several CSV files into one CSV file.

----------------------------------------------------
--------------- How To Use The Tool: ---------------
----------------------------------------------------
1- Press the "1- Select Files" button to choose the CSV files you want to combine.
\
2- Left click the blue underlined headers you want to keep.
\
3- Press the "3- Start Combining!" button.

----------------------------------------------------
-------------- Functions Of The Tool: --------------
----------------------------------------------------
- Keeps duplicate rows.
- Filters out duplicate headers.
- Accepts Arabic CSV data.
- Allows picking specific headers by left clicking, click again to remove headers.
- Upper box contains total detected headers.
- Bottom box contains headers picked.
