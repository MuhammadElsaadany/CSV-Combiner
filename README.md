# CSV-Combiner
A small tool to merge and combine several CSV files into one CSV file.

*it keeps duplicate rows, but it does filter out duplicate column headers if they share the exact same name*


--------------- How to use the tool: ---------------

1- Press the "1- Select Files" button to choose the CSV files you want to combine.

2- From the upper box in the tool, left click with your mouse on the blue underlined column headers you want to keep. (left click again to remove!)
*the 2nd box in the tool contains column headers you picked*

3- Press the "3- Start Combining!" button to combine the selected CSV files (using the column headers you picked earlier) into one CSV file.
